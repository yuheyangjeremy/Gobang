# Gobang
CSCI3100 Group B3

## Deployment
Run the following lines in terminal:
```terminal
npm install
```

## Test
Run the following lines in terminal:
```terminal
node src/server/server.js
```

Open a new terminal and run:
```terminal
npm start
```

## Settings
The server port is 3000 and the client port is 80, please use http://localhost:80/index for testing
